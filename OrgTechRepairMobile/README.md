# ВузяПринт — мобильное приложение

Полноценный порт сайта OrgTechRepair (информационная система ВузяПринт) во Flutter.

## Требования

- Flutter SDK (добавьте `flutter/bin` в PATH)
- Запущенный бэкенд OrgTechRepair (сайт) на порту 5121

## Первый запуск

1. **Создать платформенные каталоги** (если папок `android/` и `ios/` ещё нет):
   ```bash
   cd C:\Users\valpc\AndroidStudioProjects\OrgTechRepairMobile
   flutter create . --project-name orgtechrepair_mobile
   ```
2. Установить зависимости:
   ```bash
   flutter pub get
   ```
3. Запуск на эмуляторе или устройстве:
   ```bash
   flutter run
   ```

## Настройка URL API

Базовый URL задаётся в `lib/core/app_config.dart` (`kApiBaseUrl`). На экране входа можно указать адрес сервера и сохранить его (для телефона — IP ПК в Wi‑Fi и порт **5121**; без порта подставится `:5121`).

- **Эмулятор:** удобно `http://10.0.2.2:5121`.
- **Реальное устройство:** например `http://192.168.1.100:5121`.

## Функциональность

- **Главная** — приветствие и быстрая заявка на ремонт принтера (без входа).
- **Вход / Регистрация / Забыли пароль / Сброс пароля** — через API `api/Auth/*`.
- **Заявки** — список, просмотр, создание и редактирование (по ролям).
- **Клиенты, Товары, Поставщики** — список и CRUD (по ролям).
- **Продажи, Сотрудники, Запчасти** — просмотр списков (по ролям).
- **Отчёты** — продажи и заявки за период (API `api/Reports/*`, роли бухгалтер/директор/админ).
- **Администрирование** — пока заглушка в приложении (на бэкенде логи/бэкап доступны по JWT для администратора).

Меню в боковой панели отображается в зависимости от ролей пользователя (как на сайте).
