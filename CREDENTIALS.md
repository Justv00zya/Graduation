# Учетные данные для входа в систему

## Администратор по умолчанию

При первом запуске приложения автоматически создается пользователь-администратор:

- **Логин:** `admin`
- **Пароль:** `111111`
- **Роль:** Administrator (полный доступ)

## Важно!

1. **Регистр важен:** Логин должен быть введен точно как `admin` (строчными буквами)
2. **Пароль:** `111111` (шесть единиц)

## Если не получается войти:

1. Убедитесь, что база данных была создана при первом запуске
2. Проверьте логи приложения на наличие ошибок при создании пользователя
3. Если пользователь не был создан, можно:
   - Удалить базу данных и перезапустить приложение
   - Или создать пользователя вручную через раздел "Администрирование" (если есть доступ)

## Создание дополнительных пользователей

После входа как администратор, вы можете создать дополнительных пользователей через раздел **"Администрирование"** и назначить им соответствующие роли:
- Manager (Менеджер)
- Engineer (Инженер)
- Accountant (Бухгалтер)
- Director (Директор)
