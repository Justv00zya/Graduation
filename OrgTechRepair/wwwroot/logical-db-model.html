Логическая модель базы данных (OrgTechRepair / ВузяПринт)

Соответствует сущностям EF Core в OrgTechRepair.Models и таблицам Identity в той же базе. Имена таблиц по соглашению EF — во множественном числе (как DbSet). Диаграмма связей — Mermaid erDiagram (нужен интернет для CDN).
