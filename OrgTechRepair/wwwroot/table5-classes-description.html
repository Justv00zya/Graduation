Таблица 5 — Описание классов (модуль веб-приложения OrgTechRepair)

№ Наименование класса Назначение класса
1 Program Точка входа ASP.NET Core: регистрация Blazor Server, Entity Framework Core, Identity (cookie), JWT Bearer для API, CORS, маршрутизация контроллеров и Razor-компонентов, вызов инициализации БД и сидирования.
2 ApplicationDbContext Контекст базы данных EF Core: наборы сущностей (заявки, клиенты, сотрудники, товары, запчасти, продажи, заявки на склад и др.), связи и конфигурация таблиц.
3 SeedData Статический класс начального заполнения: роли (администратор, менеджер, клиент, кладовщик и др.), учётные записи демо-пользователей, справочники, тестовые заявки и данные для отчётов.
4 AccountController MVC-контроллер учётной записи: вход по логину или email и паролю (cookie), серверная регистрация клиента с antiforgery-токеном, назначение роли и вход после успешной регистрации.
5 AuthController API-контроллер аутентификации: выдача JWT при входе, регистрация клиента для мобильного приложения, смена пароля; используется мобильным клиентом и интеграциями.
6 OrdersController REST API заявок на ремонт: список и детали для авторизованных ролей, создание и изменение; анонимная быстрая заявка с главной страницы сайта; при сохранении заявки вызывается генерация PDF.
7 ClientCabinetController API личного кабинета клиента: доступ к своим заявкам и данным профиля в рамках прав пользователя с ролью Client.
8 AdminController API для администратора: просмотр и выгрузка журнала приложения (JSON, CSV), запуск резервного копирования БД и скачивание дампа.
9 PartSupplyRequestsController API заявок сервисного инженера на выдачу запчасти со склада: создание, список для кладовщика, одобрение или отклонение заявки.
10 OrderPdfService Сервис формирования PDF-документа заявки на ремонт (QuestPDF): данные клиента и заявки, сохранение файла в каталог из конфигурации OrdersArchive.
11 ClientProfileProvisioner Сервис создания или поиска карточки клиента в БД и привязки её к пользователю Identity при регистрации и входе (личный кабинет и заявки).
12 DatabaseBackupService Сервис резервного копирования базы PostgreSQL через утилиту pg_dump (путь задаётся в конфигурации).
13 RevalidatingIdentityAuthenticationStateProvider Провайдер состояния аутентификации Blazor Server: периодическая ревалидация пользователя относительно хранилища Identity, чтобы сессия соответствовала актуальным данным в БД.
