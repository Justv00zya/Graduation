## Для запуска приложения

1. Убедитесь, что установлен .NET 8.0 SDK
2. Убедитесь, что установлен SQL Server LocalDB
3. Выполните команду: `dotnet run` в папке `OrgTechRepair`
4. Откройте браузер по адресу: `https://localhost:7281` или `http://localhost:5121`
5. Войдите с учетными данными:
   - **Логин:** `admin`
   - **Пароль:** `Admin123!`
