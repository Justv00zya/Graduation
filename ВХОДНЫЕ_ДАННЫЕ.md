# Входные данные

- **Заявки на ремонт**
  - номер заявки;
  - информация о клиенте;
  - модель оборудования;
  - описание состояния оборудования;
  - описание неисправности (жалоба);
  - информация об исполнителе (сотруднике);
  - стоимость ремонта;
  - дата приёма заявки;
  - дата выполнения;
  - статус заявки (принят, в работе, выполнен, отменён).

- **Позиции заявок по запчастям**
  - информация о заявке;
  - информация о запчасти;
  - количество.

- **Клиенты**
  - ФИО или наименование организации;
  - адрес;
  - телефон.

- **Поставщики**
  - наименование;
  - адрес;
  - ИНН;
  - расчётный счёт;
  - телефон.

- **Товары**
  - код товара;
  - информация о поставщике;
  - наименование;
  - модель;
  - цена;
  - количество на складе.

- **Запчасти**
  - код запчасти;
  - информация о поставщике;
  - наименование;
  - цена;
  - количество на складе.

- **Должности**
  - наименование должности;
  - оклад.

- **Сотрудники**
  - табельный номер;
  - информация о должности;
  - фамилия;
  - имя;
  - отчество;
  - дата рождения;
  - ИНН;
  - адрес;
  - дата приёма на работу.

- **Виды работ**
  - наименование вида работ;
  - цена.

- **Работы по заявке**
  - информация о заявке;
  - информация о виде работ;
  - порядковый номер.

- **Продажи**
  - номер продажи;
  - информация о клиенте;
  - дата продажи;
  - итоговая сумма.

- **Позиции продаж**
  - информация о продаже;
  - информация о товаре;
  - количество;
  - цена за единицу;
  - сумма по позиции.

- **Пользователи (учётные записи для входа в систему)**
  - имя пользователя (логин);
  - электронная почта;
  - пароль;
  - назначенные роли.

- **Отправка быстрой заявки (форма на главной странице)**
  - ФИО или название организации;
  - телефон для связи;
  - модель принтера;
  - описание проблемы (неисправности).
